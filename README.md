# The Living Practice

A static yoga-teaching site by Ally Krupar — anatomy, philosophy, and the ancient teachings.

Plain HTML/CSS/JS, no build step. Each page is a self-contained file.

- `index.html` — landing page
- `yoga-family-tree.html` — interactive tree of yoga families mapped to the chakra system
- `yoga-muscle-diagram.html` — interactive front/back muscle diagrams
- `cosmic-banyan.html` — Vedic philosophy and text explorer (uses `tree.png`)

## Running locally

Open `index.html` in a browser, or serve the folder:

```
python -m http.server 8000
```

then visit http://localhost:8000

## Hosting

Published via GitHub Pages — no build required; the files are served as-is.
